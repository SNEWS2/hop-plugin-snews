============
Installation
============

.. contents::
   :local:

You can install hop-plugin-snews via pip or from source.

To install with pip:

.. code:: bash

   pip install -U hop-plugin-snews

To install from source:

.. code:: bash

    tar -xzf hop-plugin-snews-x.y.z.tar.gz
    cd hop-plugin-snews-x.y.z
    python setup.py install
